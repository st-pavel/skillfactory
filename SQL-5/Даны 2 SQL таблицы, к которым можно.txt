Даны 2 SQL таблицы, к которым можно обратиться как sql.matches и sql.teams

1. Таблица matches с данными о матчах
Название столбца
Содержимое столбца
id	id матча
season	сезон
date	дата матча
home_team_api_id	api_id домашней команды, ключ на таблицу teams
away_team_api_id	api_id гостевой команды, ключ на таблицу teams
home_team_goals	количество голов домашней команды
away_team_goals	количество голов гостевой команды

2.Таблица teams с данными о командах
Название столбца
Содержимое столбца
id	id команды
api_id	ключ на таблицу matches
long_name	полное название команды
short_name	сокращённое название команды

Есть уточнение - есть несколько различных команд с одинаковым полным названием.
Лучше использовать группировку по столбцу id таблицы teams, хотя этот столбец не выводится в запросе. Это необходимо для того, чтобы команды с одинаковым названием, если такие найдутся, не группировались между собой. Группировка по названию команды в данном запросе будет неверной, так как есть несколько команд с одинаковым полным названием
